# hyb_current_actual Notes

This folder contains the current real ANSYS node and topology-density exports:

- `NLIST.lis`: node coordinates.
- `export1.txt`: node-wise topology density.

The case is used by `configs/cases/hyb_current_actual.yaml` and golden regression tests. It is a software-integration baseline only; it is not enough data for formal neural-network accuracy or generalization claims.
